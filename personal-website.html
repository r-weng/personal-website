Projects
Below are some of the projects I've worked on. Click on them see more!
YouLingo | JavaScript
A dynamic web application that offers a fresh perspective on language learning by transforming YouTube videos into engaging language lessons tailored to the user's interests. YouLingo was created in 36 hours and won "Best University of Toronto Hack" at UofTHacks 12.
CollabGraph | Python
A visual representation of all the musical artists that a given artist has collaborated with over time. CollabGraph was created as my CSC111 final project and was required to utilize graphs/trees in some way.
Calculator | JavaScript, HTML/CSS
A simple calculator application. This started as a copy of the calculator app taught in SuperSimpleDev's JavaScript YouTube tutorial, but I later expanded on the app by adding my own features and a simple yet visually appealing interface.
Rock Paper Scissors | JavaScript, HTML/CSS
A rock-paper-scissors game you can play by yourself. I built this while learning how to implement JavaScript with HTML/CSS using SuperSimpleDev's YouTube tutorial. The interface and mechanics of the game are directly taken from the tutorial.
